# 局域网访问配置说明

## 快速开始

### 1. 获取本机IP地址

**Windows:**
```bash
ipconfig
```
查找 "IPv4 地址"，例如：`192.168.1.100`

**Mac/Linux:**
```bash
ifconfig
```
或
```bash
ip addr show
```

### 2. 启动服务器

**启动后端（终端1）:**
```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**启动前端（终端2）:**
```bash
cd frontend
npm run dev
```

### 3. 访问游戏

**本机访问:**
- 前端: http://localhost:3000
- 后端API: http://localhost:8000

**局域网内其他设备访问:**
- 前端: http://[您的IP地址]:3000
- 例如: http://192.168.1.100:3000

## 注意事项

1. **防火墙设置**
   - Windows: 需要在防火墙中允许端口3000和8000
   - Mac: 系统偏好设置 → 安全性与隐私 → 防火墙
   - Linux: 使用 `ufw allow 3000` 和 `ufw allow 8000`

2. **网络要求**
   - 确保所有设备在同一局域网内
   - 确保路由器没有阻止设备间通信

3. **IP地址变化**
   - 如果IP地址是动态分配的，重启后可能会变化
   - 建议在路由器中为您的电脑设置静态IP

4. **测试连接**
   - 在其他设备上先访问 http://[您的IP]:8000/docs 测试后端是否可访问
   - 然后再访问前端地址

## 故障排查

### 无法访问？
1. 检查防火墙设置
2. 检查IP地址是否正确
3. 检查后端和前端是否都在运行
4. 尝试ping您的IP地址

### WebSocket连接失败？
- 确保后端运行在 0.0.0.0:8000
- 检查浏览器控制台的错误信息
- 确保没有代理服务器干扰
